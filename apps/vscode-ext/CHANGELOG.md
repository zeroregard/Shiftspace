# Changelog

All notable changes to Shiftspace will be documented in this file.

## [0.1.34] — 2026-04-06

### Added

- Jump-to-line support for annotation badges
- Status bar insights, resizable file list, animated loader
- Bundled VSCode themes (Shiftspace Dark, Shiftspace Light)

### Changed

- Button styling cleanup and linting pass
- Modernized color palette to 3-layer depth system

### Fixed

- Stability fixes across the board

## [0.1.33] — 2026-04

### Added

- "All files" diff mode to browse all tracked repository files
- Playwright component tests for core UI components
- Deterministic seeded PRNG for stable E2E screenshots

### Fixed

- Deduplicate partially-staged files in tree view

## [0.1.32] — 2026-04

### Fixed

- Render slim grove view in sidebar instead of redirect

## [0.1.31] — 2026-03

### Added

- Sidebar view with slim worktree cards

### Fixed

- Status bar improvements

## [0.1.30] — 2026-03

### Added

- Command execution security with input validation
